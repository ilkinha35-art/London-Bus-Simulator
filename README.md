# London Bus Simulator

Um simulador realista de ônibus londrinos inspirado no sistema de transporte da TfL.  
Este projeto é open-source e feito para rodar em **Unity Engine**.

---

## 🚍 Funcionalidades
- Ônibus clássicos de Londres (Routemaster, Enviro400).
- Rotas baseadas em dados reais da TfL.
- Sistema de tráfego dinâmico com carros e pedestres.
- Clima e horário dinâmicos (dia/noite, chuva, sol).
- Sistema iBus com anúncios de rota.
